# AutomatizacionSelenium

Proyecto de automatización web con **Python + Selenium** para validar flujos funcionales de la página de **Credifamilia CDT**.

## Objetivo

Este repositorio contiene scripts de automatización orientados a validar tres procesos principales del portal:

- Simulación de CDT
- Contacto con asesor por WhatsApp
- Registro de apertura de CDT

El objetivo es comprobar que los formularios, componentes dinámicos y flujos visibles para el usuario respondan correctamente ante datos válidos.
